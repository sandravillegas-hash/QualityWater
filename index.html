¿Qué es el IRCA?

El Índice de Riesgo de la Calidad del Agua es el indicador oficial en Colombia para evaluar la seguridad del agua para consumo humano.

Base Legal

Decreto 1575 de 2007

Establece el sistema para la protección y control de la calidad del agua

Resolución 2115 de 2007

Define los parámetros y valores máximos permitidos en agua potable

Parámetros Evaluados

Físicos

  • • Turbidez
  • • Color
  • • Olor
  • • Sabor

Químicos

  • • pH
  • • Cloro residual
  • • Metales pesados
  • • Nitratos

Microbiológicos

  • • Coliformes totales
  • • Coliformes fecales
  • • E. coli

Pesticidas

  • • Organofosforados
  • • Carbamatos
  • • Triazinas
  • • Piretroides

Interpretación del IRCA

El IRCA refleja la probabilidad de que el agua cause enfermedades de origen hídrico si no cumple con los estándares de calidad. Se expresa en porcentaje (0-100%) y clasifica el agua en niveles desde "Sin Riesgo" hasta "Inviable Sanitariamente".

Funcionalidades

Características Principales

Una herramienta completa diseñada para investigadores, funcionarios públicos y ciudadanos interesados en la transparencia de datos hídricos.

Filtros Interactivos

Filtra registros por año (2007-2024), departamento y municipio para un análisis granular y específico de los datos.

KPIs Dinámicos

Visualización automática de promedios IRCA (General, Urbano, Rural) y conteo de mediciones actualizados en tiempo real.

Gráficos Plotly

Evolución temporal, distribución de riesgos, Top 10 municipios críticos y análisis de dispersión comparativo.

Base de Datos Dual

Soporte flexible para MySQL (producción) o SQLite (local) configurado mediante variables de entorno.

Exportación CSV

Descarga la información filtrada directamente a archivos .csv para reportes externos o análisis adicionales.

Diseño Responsivo

Interfaz adaptativa que funciona perfectamente en escritorio, tabletas y dispositivos móviles.

Instalación y Configuración

Sigue estos pasos para levantar el dashboard en tu entorno local en menos de 5 minutos.

1

Clonar Repositorio

Obtén el código fuente

Consola
git clone https://github.com/sandravillegas-hash/QualityWater.git
2

Instalar Dependencias

Requiere Python 3.9+

Consola
pip install -r requirements.txt
streamlit pandas plotly sqlalchemy mysql-connector-python python-dotenv
3

Configurar .env

Define tu conexión

Crea un archivo .env en la raíz del proyecto: 

DB_TYPE=sqlite
DB_HOST=127.0.0.1
DB_USER=root
DB_PASSWORD=tu_contraseña
DB_NAME=quality_water
DB_PORT=3306

Recomendado: Si usas DB_TYPE=sqlite, el proyecto leerá automáticamente el archivo quality_water.db sin necesidad de configurar usuario o contraseña. Ideal para desarrollo local.

4

Ejecutar Dashboard

¡Listo para usar!

Consola
streamlit run https://qualitywater-smv.streamlit.app/

La aplicación se abrirá automáticamente en:

https://qualitywater-smv.streamlit.app/
